# Web Portfolio

Personal portfolio of Anthony Rosado — built with [Astro](https://astro.build)
and [Tailwind CSS v4](https://tailwindcss.com), deployed to GitHub Pages
at [anthonyrosado.dev](https://anthonyrosado.dev).

## Tech stack

- **Astro 5** — static site generator
- **Tailwind CSS v4** — styling
- **@astrojs/sitemap** — automatic sitemap generation
- **GitHub Pages** — hosting
